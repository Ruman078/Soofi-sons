# Soofi Sons & Co. — Website

A single-file static site (`index.html`) — no build step needed.

## Deploy to Vercel (fastest way, no coding required)

**Option A — Drag and drop:**
1. Go to https://vercel.com and sign up / log in (free).
2. Click **Add New → Project**.
3. Choose **"Upload"** (or drag the `soofi-sons-website` folder onto the dashboard).
4. Vercel detects it as a static site — click **Deploy**.
5. You'll get a live URL like `soofi-sons.vercel.app` in under a minute.

**Option B — Via GitHub (recommended for future edits):**
1. Create a new repo on GitHub, upload `index.html` (and this README).
2. On vercel.com, click **Add New → Project → Import Git Repository**, pick that repo.
3. Leave all settings default (it's static HTML) and click **Deploy**.
4. Every time you push a change to GitHub, Vercel redeploys automatically.

## Custom domain
Once deployed, go to your Vercel project → **Settings → Domains** and add your own domain (e.g. `soofisons.pk`) if you buy one.
